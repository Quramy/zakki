# 5/2

## reg-suit のメンテと GCP

自分のクレカ更新したときに Google の課金情報を更新するのを忘れていたせいで、抱えている GCP Project のいくつかが停止されていた。
このせいで、reg-suit の開発用に使っている GCS にアクセスできなくなり、CI が全落ちしていて reg-suit の Deploy に支障をきたしていたため、アカウントを正常化して Fail していたいくつかの PR を re-run した。
